# Whist Online

A multiplayer Nomination Whist card game built with ASP.NET Core and React.

## Stack
- **Backend:** ASP.NET Core Web API (C#)
- **Frontend:** React + TypeScript
- **Database:** PostgreSQL

## Project Structure
```
whist-online/
  backend/    ← ASP.NET Core Web API
  frontend/   ← React + TypeScript
```
